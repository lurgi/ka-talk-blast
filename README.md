# 🚀KA TAKL BLAST!

카톡으로 대량의 메세지를 보낼 수 있는 기능을 제공합니다.

## 사용자 흐름

1. 카카오 로그인
2. 메세지 폼을 선택 그리고 메세지를 만든다.
3. 보내는 친구를 선택한다.
4. 메세지를 보낸다.

## 기능 목록

- [x] 카카오 로그인 구현
  - [x] 카카오 token 받아오기 및 쿠키 저장
- [x] 카카오 메세지폼 만들기

  - [x] 나에게 보내기
  - [x] 메세지 폼 UI

### 불가능 ❌❌❌

- ❌ 카카오 친구 목록 불러오기 구현
- ❌ 카카오 친구 에게 메세지 보내기 구현
  카카오 앱에 등록된 친구가 아니면 불러오기 불가능. 즉 하나의 Sender에 직접적으로 등록해야만 메세지를 보낼 수 있다.
